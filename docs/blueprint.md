# **App Name**: Inmobiliaria AI CRM

## Core Features:

- Client Management: Add, edit, and organize client information (name, contact details, preferences, etc.).
- Property Tracking: Manage properties, associating them with clients.
- Lead Scoring: Score the user based on their interest.
- Interaction History: Log all interactions with clients (calls, emails, meetings) for a complete communication history.
- AI-Powered Response Generation: Generate automated responses to client inquiries using AI; provide the user with the generated responses and ability to edit them before sending. The LLM will use information about the client from its internal toolset.
- Contact through Email and SMS: Send emails or SMS directly from the platform to your clients.
- Reminder setting: Allow users to set reminders for tasks with clients.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for trust and professionalism.
- Background color: Light blue (#E8EAF6), a very light tint of the primary.
- Accent color: Violet (#7E57C2) for highlights and call-to-action elements.
- Headline font: 'Space Grotesk' sans-serif, to give the CRM application a modern, trustworthy, computerized feel.
- Body font: 'Inter' sans-serif; since this app has short form content, using two fonts is unnecessary, and 'Inter' will work in either role.
- Use professional and clear icons representing various real estate actions (e.g., property search, client contact, etc.).
- Use subtle transition animations to give smooth feedback, as users interact with UI elements such as button presses or tab changes.
